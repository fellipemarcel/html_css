Marca Texto

Podemos também marcar uma parte do texto que seja muito relevante para o leitor.

E ainda, podemos usar o parâmetro style para trocar a cor do marca-texto. Style é referente a CSS. Logo, estamos usando CSS dentro de apenas uma tag, que é a mark.

Para trocar a cor de todas as Mark, é melhor criar uma tag style dentro da head, aplicando o estilo CSS correto.

Big e Small

A tag <big> deixou de ser utilizada, mas a tag <small> ainda pode ser utilizada dessa forma.

Texto Deletado

O texto deletado serve para você mostrar para o leitor que o texto ainda está lá, porém, não deve ser levado em consideração.

Texto Inserido

O texto inserido é o contrário do deletado. Este serve para indicar ao leitor que uma determinada parte do texto é muito importante e deve-se prestar atenção nela.
